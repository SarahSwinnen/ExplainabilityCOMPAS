# Explaining COMPAS Recidivism Racial Bias

This dataset used in this project is from:
https://www.kaggle.com/datasets/danofer/compass?select=cox-violent-parsed_filt.csv

# Dataset Features
The features that we will include are:
* is_recid (target)
* age
* sex
* race
* juv_fel_count
* juv_misd_count
* juv_other_count
* c_charge_degree
* priors_count
